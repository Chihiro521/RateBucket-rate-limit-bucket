# Nahida UI 资产需求列表

为了实现高保真度（High-Fidelity）的纳西妲风格 UI，以下元素建议生成为 **透明背景 (PNG/WebP)** 或 **矢量 (SVG)** 素材，而非纯 CSS 实现。

## 1. 核心装饰 (High Priority)
- **chibi-nahida-peeking.webp**: 趴在面板边缘探头的纳西妲小人（用于页眉装饰）。
- **ornate-footer-decoration.webp**: 底部中央的大型装饰件（宝石+藤蔓，用于页脚）。
- **vine-corners.webp**: 四角的装饰藤蔓（可以是一组四个，也可以是一个，通过旋转使用）。
- **divider-ornate-line.webp**: 带有中央小花朵或宝石装饰的水平分割线。

## 2. 图标类 (Medium Priority)
- **icon-shield-nahida.webp**: 带有纳西妲四叶草标志的护盾图标（用于预警或安全检测）。
- **icon-clover-action.webp**: 精致的四叶草按钮图标（用于代替简单的 Unicode 字符）。
- **icon-leaf-bullet.webp**: 小叶子图标（用于列表项或小标题前缀）。

## 3. 细节增强 (Polish)
- **progress-leaf-tip.webp**: 进度条末端的小叶子装饰（比 SVG Mask 效果更真实，带阴影和高光）。
- **background-vines-pattern.webp**: 极淡的底纹（用于面板背景，增加纸张或自然感）。
- **gem-emerald-square.webp**: 翠绿色方形宝石图标（用于状态显示）。

---
**提示**：生成素材时请保持色调一致，参考：
- 墨绿色 (#31461f)
- 嫩绿色 (#6f9c37)
- 香槟金 (#c8a94f)
- 象牙白 (#fffdf3)
